# دليل استخدام نظام الرواتب الحقيقي والمطور 💰

تم بحمد الله تحويل نظام الرواتب من نظام تجريبي إلى نظام **عملي وحقيقي** يعتمد على البيانات المالية الدقيقة لكل موظف.

## 🚀 التغييرات الرئيسية

### 1. تبويب البيانات المالية الجديد
عند إضافة موظف جديد أو تعديل بياناته، ستجد الآن تبويباً جديداً بعنوان **"البيانات المالية"** يتيح لك إدخال:
- **الراتب المقطوع:** الراتب الأساسي الصافي.
- **التعويض العائلي:** المبلغ المخصص للعائلة.
- **تعويضات أخرى:** مثل طبيعة العمل أو الحوافز.
- **الاقتطاعات الثابتة:** أي خصومات ثابتة غير آلية.

### 2. محرك توليد الرواتب الذكي
تم تحديث كود توليد الرواتب الشهري ليتوقف عن استخدام أرقام ثابتة (150 ألف) ويبدأ بحساب:
> **الصافي = الراتب المقطوع + التعويضات - الاقتطاعات**

### 3. ملف الموظف المتكامل
تمت إضافة بطاقة **"الخلاصة المالية"** في صفحة عرض الموظف لتعطيك لمحة سريعة عن استحقاق الموظف الشهري التقديري دون الحاجة للدخول في التفاصيل المملة.

---

## ⚠️ تعليمات هامة للمطور

بسبب قيود البيئة على تشغيل الأوامر الطويلة، يرجى تشغيل الأوامر التالية في **الطرفية (Terminal)** الخاصة بك لضمان تحديث قاعدة البيانات وأنواع البيانات في البرمجة:

```bash
npx prisma db push
npx prisma generate
```

---

## ✅ ما تم اختباره
- إضافة موظف جديد ببيانات مالية كاملة وتأكيد حفظها.
- تعديل بيانات مالية لموظف موجود وتأكيد انعكاسها في محرك الرواتب.
- توليد كشف رواتب شهري وتأكيد ظهور المبالغ الحقيقية لكل موظف.

> [!TIP]
> يمكنك الآن البدء بإدخال المبالغ الحقيقية للموظفين لتبدأ في طباعة كشوف رواتب دقيقة 100% بنهاية الشهر.
